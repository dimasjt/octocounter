# Octocounter


## Installation

$ gem install octocounter

## Usage

$ octocounter path/to/directory

## Result

| Files                  | Content     | Count |
|------------------------|-------------|-------|
| test/data/A/content2<br>test/data/B/D/content3<br>test/data/B/content1   | bcdef       | 3     |
| test/data/B/D/diff     | bcdefghijkl | 1     |
